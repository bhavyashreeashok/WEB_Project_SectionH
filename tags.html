Introduction to Basic HTML Tags

    Basic Structure Tags

  • !DOCTYPE html tag : Declares the document type as HTML5.
    •
  • html tag : Root element containing the entire HTML document.
    •
  • head tag : Contains metadata, links, and document settings.
    •
  • title tag : Sets the title in the browser tab.
    •
  • body tag : Main container for visible web page content.
    •

    Heading and Formatting Tags

  • h1 to h6 tags : creates headings, where h1 tag is the largest.
  • p tag : Creates paragraphs.
  • b tag : Makes the text bold.
  • i tag : Italicises the text.
  • u tag : Underlines the text.
  • strong tag : Makes the text bold as well. It is an alternative for Bold tag.
  • em tag : Italicises the text as well. It is an alternative for Italics tag.
  • br tag : Breaks the line and goes to the next line.
  • hr tag : Adds a horizontal rule.
    •

    List Tags

  • ul tag : Creates an Unordered (bulleted) list.
  • ol tag : Creates an Ordered (numbered) list.
  • li tag : Used to list items within unordered and ordered lists.
  • dl tag : Creates a Description list.
    1. dt tag : Creates the terms in a description list.
    2. dd tag : Used to write definitions for the terms in a description list.

    Hyperlink and Media Tags

  • a tag : Called as Anchor tag. Its creates hyperlinks that link different web pages to the current web page.
  • img tag : Allows to add images from the file explorer using the src tag to specify the path of the image.
  • audio tag : Adds audio content to the page, using the src tag to specify the path of the audio.
  • video tag : Adds video content to the page, using the src tag to specify the path of the video.
  • source tag : Specifies the media sources.
  • iframe tag : Inline frame, embeds external content directly into a webpage.

    Form Tags

  • form tag : Creates Forms in a web page.
  • input tag : Adds a form input field (text, checkbox, radio, etc.).
  • textarea tag : Used to give multiline text input by the users, like feedbacks.
  • button tag : Creates a clickable button.
  • label tag : Creates labels for form elements.
  • select tag : Creates a Drop-down list to select from.
  • option tag : Adds options in a drop-down list.
  • fieldset tag : Groups related elements within a form.
  • legend tag : Creates a captions for a fieldset in a form.
  • datalist tag : Provides a list of predefined options, like slect tag, but here the user can also type his/ her own inputs

    Table Tags

  • table tag : Creates a table element in the web page.
  • tr tag : Creates rows in a table element.
  • td tag : Adds data in a cell of the table element.
  • th tag : Adds headings in a cell of the table element.
  • thead tag : Groups the header content of a table element.
  • tbody tag : Contains the main data rows of the table.
  • tfoot tag : Groups footer content, often used for totals or summaries.
  • caption tag : Adds captions to a table element.
    •

    Semantic Tags

  • header tag : Creates the document or section header.
  • footer tag : Creates Footer for a section or an entire page.
  • nav tag : Creates navigation links.
  • article tag : Makes the content Self-contained.
  • section tag : Creates a Generic section of a document.
  • aside tag : Used to define content that is indirectly related to the main content of a page, its like a a sidebar or a tangential note
  • main tag : It contains the Main content area.
  • figure tag : Creates an image with an optional caption.
  • figcaption tag : Adds caption for the image in the figure tag.

    Miscellaneous Tags

  • span tag : It is a generic inline container used to group and style parts of text or other inline elements, whose power comes from using it with CSS or JAVA.
  • div tag : It's a block-level container that helps organize and structure content on a webpage.
  • mark tag : Highlights the text in default yellow colour.
  • abbr tag : Used to represent an abbreviation or acronym, providing a full description or expansion when needed.
  • address tag : Used to define contact information for a person, organization, or website.
  • blockquote tag : Block-level quotation, used to indicate a section of text that is quoted from another source.
  • q tag : Inline quotation, used to mark short, inline quotations within a sentence.
  • code tag : Used to represent a piece of computer code or programming-related text.
  • pre tag : Used to display preformatted text,content inside it is shown exactly as written, including spaces, tabs, and line breaks.
  • small tag : Used to render text in a smaller font size than the surrounding content.
  • sup tag : Superscript, used to add text above the usual line, like in squared numbers.
  • sub tag : Subscript, used to add text below the usual line, like in chemical formulas.
  • del tag : Displays the deleted text by strikethrough.
  • ins tag : Adds an inserted text, underlines the text as well, thereby can be called as an alternative for the u tag.
  • wbr tag : Word break opportunity, suggests a line-break opportunity within text.

    Self-Closing Tags

  • br tag
  • hr tag
  • img tag : Embeds an image; requires src and alt attributes.
  • input tag
  • meta tag : Provides metadata like charset, description, author.
  • link tag : Links external resources like CSS stylesheets or favicons.
  • source tag
  • area tag : Defines clickable areas in an image map.
  • base tag : Sets the base URL for relative links in the document.
  • col tag : Specifies column properties in a table.
  • embed tag : Embeds external content like videos or interactive media.
  • track tag : Adds subtitles or captions to media.
  • wbr tag
